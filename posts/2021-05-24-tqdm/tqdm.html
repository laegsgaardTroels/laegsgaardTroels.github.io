Tqdm

Python
2021-05-24
[View source ]

tqdm is a nice Python module for creating a progressbar when processing a list of items.

It can be used in a basic for loop:

from tqdm import tqdm

xs = [1, 2, 3]
ys = []
with tqdm(total=len(xs)) as pbar:
    for x in xs:
        ys.append(x ** 2)
        pbar.update()

And with multiprocessing as well:

import multiprocessing
from tqdm import tqdm

# The number of parallel processes to run.
PARALLELISM = 2

def task(x):
     return x ** 2

xs = [1, 2, 3]
ys = []
with tqdm(total=len(xs)) as pbar:
    with multiprocessing.Pool(processes=PARALLELISM) as pool:
        for y in pool.imap(task, xs):
            ys.append(y)
            pbar.update()
